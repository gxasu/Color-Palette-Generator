# ユビキタス言語辞書

Writing Supervisor が管理するプロジェクト共通用語の定義。
すべての UI テキスト、コード内コメント、ドキュメントはこの辞書に準拠する。

---

## 色とパレット

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| パレット | Palette | 1つのベースカラーから生成される色のスケール全体。複数のモードを持つ。 |
| カラースケール | Color Scale | パレットの別名。明度または透明度の段階的なグラデーション。 |
| ステップ | Step | スケール内の個々の色。100, 200, 300... のような名前を持つ。 |
| ステップ名 | Step Name | 各ステップの識別子（デフォルト: 100〜）。ユーザーが編集可能。 |
| ベースカラー | Base Color | パレット生成の起点となる色。スウォッチ上に「ベース」バッジで表示。 |
| カラー数 | Color Count | 1つのパレットに含まれるステップの数（2〜20）。 |

## モードとコレクション

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| モード | Mode | 同一パレット構造に対する色のバリエーション。Figma Variables の Mode に対応。 |
| コレクション | Collection | 複数のパレットをまとめるグループ。Figma Variables の Collection に対応。 |
| コレクション名 | Collection Name | エクスポートファイル名の基になる名前。 |

## 色空間と値

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| 明度 | Lightness (L) | OKLCH 色空間における知覚的明度。0（黒）〜1（白）。 |
| 彩度 | Chroma (C) | OKLCH 色空間における色の鮮やかさ。0（無彩色）〜最大値（色相依存）。 |
| 色相 | Hue (h) | OKLCH 色空間における色味の角度。0°〜360°。 |
| 透明度 | Alpha | 色の不透明度。0（完全透明）〜1（完全不透明）。 |
| 明度カーブ | Lightness Curve | ステップ間の明度分布を制御するカーブパラメータ。-1.0〜1.0。 |

## パレットタイプ

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| (標準パレット) | OKLCH Palette (`paletteType: 'oklch'`) | ベースカラーの色相・彩度を維持し、明度を段階的に変化させるパレット。 |
| Alpha パレット | Alpha Palette (`paletteType: 'alpha'`) | 同一 HEX 色の透明度を段階的に変化させるパレット。 |

## テーマと背景

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| テーマ | Theme | アプリの外観テーマ。ライト / ダーク / システム の3択。`data-theme` 属性で制御。 |
| ライト背景 | Light Background (`lightBg`) | ライトモードのコントラスト比算出に使用する背景色（デフォルト: `#ffffff`）。 |
| ダーク背景 | Dark Background (`darkBg`) | ダークモードのコントラスト比算出に使用する背景色（デフォルト: `#1a1a1a`）。 |
| 背景プレビュー | Background Preview | スウォッチ表示領域の背景切り替え。ライト / ダーク の2択。 |

## コントラスト

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| コントラスト比 | Contrast Ratio | WCAG 2.1 に基づく2色間の輝度比。1:1〜21:1 の範囲。 |
| コントラストレベル | Contrast Level | コントラスト比に基づく WCAG 適合レベル。AAA (≥7), AA (≥4.5), A (≥3), Fail (<3)。 |

## UI セクション

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| パレット一覧 | Palette List | 左パネル。作成済みパレットのリスト。 |
| カラースウォッチ | Color Swatches | 中央パネル上部。パレットの全ステップを視覚的に表示。 |
| 明度チャート | Lightness Chart | 中央パネル中部。明度（または透明度）の折れ線グラフ。ドラッグ編集可能。 |
| 透明度チャート | Alpha Chart | Alpha パレット用のチャート。透明度の折れ線グラフ。 |
| コントラスト比 | Contrast Ratio | 中央パネル下部。各ステップとライト/ダーク背景のコントラスト比（WCAG 2.1）。 |
| ベースカラー設定 | Base Color | 右パネル上部。ベースカラーのピッカーと HEX 入力。 |
| パレット設定 | Palette Settings | 右パネル中部。カラー数、明度カーブ、背景色の設定。 |

## アクション

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| 追加 | Add (createPalette) | 新しい OKLCH パレットを作成する。 |
| Alpha | Add Alpha (createAlphaPalette) | 新しい Alpha パレットを作成する。 |
| インポート | Import | JSON ファイルからパレットを読み込む。 |
| エクスポート | Export | パレットを Figma Variables DTCG JSON ファイルとして出力する。 |
| ライト / ダーク | Light / Dark Preview | スウォッチの背景プレビューを切り替える。 |

## フィードバック

| 用語 (日本語 UI) | 用語 (英語 / コード) | 定義 |
|---|---|---|
| スナックバー | Snackbar | 操作結果を通知する一時的なメッセージ。M3 Snackbar パターン。4秒後に自動消去。 |

## 技術用語

| 用語 | 定義 |
|---|---|
| OKLCH | 知覚均等色空間。人間の色知覚に基づいた Lightness, Chroma, Hue の3軸。 |
| DTCG | Design Token Community Group。デザイントークンの標準フォーマット仕様。 |
| Gamut Mapping | sRGB 色域外の色を色域内に収束させる処理。二分探索で Chroma を減少。 |
| Figma Variables | Figma のデザイン変数システム。色、数値等を Mode 付きで管理。 |
| M3 / Material Design 3 | Google のデザインシステム第3版。本アプリの UI フレームワーク。 |
| PWA | Progressive Web App。オフライン動作、ホーム画面追加が可能なウェブアプリ。 |
| HEX | 16進数カラーコード。`#` + 6桁（例: `#ff0000`）。sRGB 色空間の表現形式。 |
| sRGB | Standard RGB。Web およびディスプレイの標準色空間。 |
| タッチターゲット | Touch Target。タッチ操作可能な領域。M3 ガイドラインでは最小 48dp。 |
| フォーカスリング | Focus Ring。キーボードフォーカス時の視覚的インジケーター。`:focus-visible` で実装。 |
| デバウンス | Debounce。連続イベントの最後の発火のみを処理する最適化手法。 |
| サニタイズ | Sanitize。外部入力値を安全な形式に変換・検証する処理。XSS 防止に必須。 |
